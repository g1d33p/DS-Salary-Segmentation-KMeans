# Data Science Salary Segmentation: K-Means Clustering

## 📌 Executive Summary
This project applies unsupervised machine learning (K-Means) to the Data Science Salaries dataset. By iterating through Silhouette scores, the dataset was segmented into distinct profiles based on experience, location, and employment type to identify structural compensation groupings within the industry.

## 🛠️ Tools & Methodology
* **Stack:** Python (Pandas, Scikit-Learn)
* **Methodology:** MinMax Scaling, K-Means Clustering, Silhouette Score Optimization.

## 📈 Key Insights
* Determined that a 9-cluster configuration yielded the optimal Silhouette score (0.41) for segmenting data science roles.
* Isolated specific clusters representing remote-heavy roles vs. strictly full-time, US-based roles, allowing for granular compensation benchmarking.
